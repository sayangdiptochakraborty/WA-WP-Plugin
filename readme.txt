Description: A simple click to chat WhatsApp Plugin for WordPress. The Plugin can be placed in 5 different regions of the screen - top right, top left, bottom right, bottom left, and middle right. When the user clicks on the button a new tab redirects to WhatsApp Web if the user is on PC, or starts the WhatsApp Application if the user is on mobile.
Installation: Follow Basic Installation procedure of any Plugin.
